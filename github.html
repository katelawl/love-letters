Love Letter

Dearest Github,

You are perfect for me because I can upload files to you and retrieve them from any computer. I first fell in love with you when I typed github.com and set up an account. That love blossomed when I first created a repository by clicking new repository and you told me exactly what to do to share my loving files with you. I can go into Bash and type git remote add origin https://github.com/’username’/’reponame’.git and git push -u origin master to upload a repository from my computer to you. You always keep my files safe. You are also very open about our relationship. Everyone can see my code and the commit we made together. You are kind and generous especially when you helped Professor Schaal’s lady friend get a job. We also have a very open relationship because you help me connect with other projects from people who upload their code. I can easily fork someone’s code and add to it. Thank you for keeping me connected to my code and other talented programmers code.

Love,

Kate Lawlor
